✱ How to add yourself as a contact

For example, to encrypt your cryptocurrency wallet's passphrase or personal documents, a picture...

  1. Open the file named contacts.js in the inc folder using any text editor.
  2. Look for the line that says: Myself PASTE_YOUR_PUBLIC_KEY_HERE.
  3. Replace PASTE_YOUR_PUBLIC_KEY_HERE with your public key.
    You can find it in the keys.js file on line 4.
  4. Save the file and close it.

✱ How to add someone else as a contact

  1. Open the same contacts.js file using a text editor.
  2. Under the line that says Myself, create a new line for each contact you want to add.
  3. Write your contact's name (letters, numbers, or underscores only : no spaces), followed by a space, then THEIR PUBLIC KEY. Each contact should be on a new line.
  4. Example:
            const contactsData = `
            Global PZ_-W-Kb2neUuzqQAMxyWuIORdJ28gjriOk5azgWj1A
            Myself XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
            JohnDoe HIS_PUBLIC_KEY
            JaneSmith HER_PUBLIC_KEY
            ThirdContact_I_canUseUnderscores_but_no_spaces_in_name CONTACT_PUBLIC_KEY
            `;
  1. Make sure you don't accidentally delete the last line of code: `;
  2. Save and close the file. You should now see your new contacts in the list when encrypting a message.

✱ How to remove a contact

  1. Open the contacts.js file using a text editor.
  2. Delete the entire line corresponding to the contact you want to remove.
  3. Save the file and close it.
