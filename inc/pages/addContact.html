ADD NEW CONTACTS

✱ How to add yourself as a contact

(For example, to encrypt your cryptocurrency wallet's passphrase or personal documents, a picture...)

  • 1. Open the file named "contacts.js" in the "inc" folder using any text editor (like Notepad).
  • 2. Look for the line that says: Myself XXXXX .
  • 3. Replace XXXXX with your public key. You can find it in the "keys.js" file, on line 4.
  • 4. Save the file and close it.

✱ How to add someone else as a contact

  • 1. Open the same "contacts.js" file using a text editor.
  • 2. Under the line N°3 that says "Myself," create a new line for each contact you want to add.
  • Write your contact's name (no spaces in the name itself), followed by a space, then THEIR PUBLIC KEY. Each contact should be on a new line.

  • Example:
    • const contactsData = `
    Global PZ_-W-Kb2neUuzqQAMxyWuIORdJ28gjriOk5azgWj1A
    Myself XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
    JohnDoe PUBLIC_KEY
    JaneSmith PUBLIC_KEY
    This_is_my_3rd_contact_I_can_use_underscore_in_the_name_but_no_space PUBLIC_KEY
    `;
  • 4. Make sure you don't accidentally delete the last line of code `;
  • 5. Save and close the file. You should now see your new contacts in the list for encrypting a message.
