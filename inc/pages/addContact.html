✱ How to add yourself as a contact

You can add yourself as a contact to encrypt private files or notes, for example, a password, wallet phrase, personal photo, or any document you want to keep encrypted with your own public key.

  1. Open the file named contacts.js, located in the inc folder of your D-Crypt installation.
    Use any text editor (for example: Notepad, Notepad++, or VS Code).
  2. Find this line inside the file:
    Myself PASTE_YOUR_PUBLIC_KEY_HERE
  3. Replace PASTE_YOUR_PUBLIC_KEY_HERE with your actual public key.
    You can find your public key inside the keys.js file, it is located on line 4, next to publicKeyBase64.
  4. After replacing it, save the file and close your text editor.
    Result: D-Crypt will now recognize you as a valid contact when encrypting files.

✱ How to add someone else as a contact

You can add one or more people to your contact list to send them encrypted messages or files. Each contact must be written on a new line in the contacts.js file.

  1. Open the contacts.js file (in the inc folder) using a text editor.
  2. Below the line that says Myself, create one new line per contact you want to add.
  3. For each contact, write:
    ContactName TheirPublicKey
    Important: The contact’s name can contain only letters, numbers, or underscores (_) no spaces are allowed.
  4. Example:
const contactsData = `
Global PZ_-W-Kb2neUuzqQAMxyWuIORdJ28gjriOk5azgWj1A
Myself XXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXXX
JohnDoe HIS_PUBLIC_KEY
JaneSmith HER_PUBLIC_KEY
ThirdContact_I_canUse_Underscores_but_not_spaces CONTACT_PUBLIC_KEY
`;
  1. Make sure you keep the last line of code at the bottom of the file:
    `;
    This character closes the data block and is required for the file to work properly.
  2. Save the file and close your editor.
    Result: All the contacts you’ve added will now appear automatically in the contact list when you encrypt messages or files.

✱ How to remove a contact

If you no longer want a contact to appear in your D-Crypt list, you can safely remove them by editing the same contacts.js file.

  1. Open the contacts.js file using a text editor.
  2. Locate the line containing that contact’s name and public key.
  3. Delete the entire line for that contact.
  4. Save the file and close it.

Tip: You can always re-add the same contact later by following the steps above.
