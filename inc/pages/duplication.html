D-Crypt Duplication

How to duplicate D-Crypt step by step:
  • 1. Make a copy of your entire D-Crypt folder.
  • 2. On the copy folder, open D-Crypt normaly through your web browser
  • 3. Go to "duplicate D-Crypt" in the menu, copy the provided code below.
  • 4. Paste and decrypt it through the "Decrypt" option in the menu
  • 5. Download the archived file
  • 6. Place the archived file at the root of the copy folder where you have "index.html, LICENCE and README".
  • 7. Go to "inc" folder and delete "contacts.js" and "keys.js".
  • 8. Return to the root and extract the archived file here (it will automatically create a new "contacts.js" and "keys.js" file inside "inc" folder without your own and private informations)
  • 9. Delete the archived file on the root.
  • 10. You can now copy the folder on a USB drive to share it.
